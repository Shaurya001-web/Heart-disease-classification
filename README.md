# 🫀 Heart Disease Classification using PyCaret

---

This project builds a machine learning model to predict the presence of heart disease using patient medical data from the UCI Heart Disease dataset.

The model is trained using Logistic Regression through the PyCaret framework, which simplifies the machine learning workflow.

---

## 📊 Project Overview

Heart disease is one of the leading causes of death worldwide. Early detection can significantly improve treatment outcomes.

This project demonstrates how supervised machine learning can be used to classify patients into:

- **0 → No Heart Disease**
- **1 → Heart Disease Present**

---

## ⚙️ Technologies Used

- Python  
- Pandas  
- PyCaret  
- Logistic Regression  

---

## 🧠 Machine Learning Concepts Used

- Supervised Learning  
- Binary Classification  
- Data Preprocessing  
- Model Training  
- Model Evaluation  

---

## 📂 Dataset

The project uses the **UCI Heart Disease Dataset**, which contains:

- 300+ patient records  
- 13 medical attributes  
- 1 target variable indicating disease presence  

### Key Features Include

- Age  
- Sex  
- Chest pain type  
- Blood pressure  
- Cholesterol  
- Maximum heart rate  
- Exercise-induced angina  
- ST depression  

---

## 🚀 Workflow

1. Load dataset  
2. Clean and preprocess data  
3. Handle categorical variables  
4. Train Logistic Regression model  
5. Evaluate model performance  

---

## 🎯 Outcome

The trained model predicts whether a patient is likely to have heart disease based on medical attributes.

This project demonstrates a complete machine learning pipeline from data preprocessing to model training.
